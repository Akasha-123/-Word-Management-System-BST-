 BST Word Management System

A C++ project based on the Binary Search Tree (BST) data structure for storing and managing words.

Features:
- Insert words from a paragraph
- Search words
- Delete words
- In-order, Pre-order, and Post-order traversal
- Find tree height
- Find longest and shortest word

Technologies:
- C++
- OOP
- Binary Search Tree (BST)

 Files:
- `main.cpp`
- `BST.cpp`
- `BST.h`
- `Node.h`

 How to Run:
Compile and run the project using Visual Studio or VS Code.
